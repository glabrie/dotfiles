This file tangles into ~fastfetch.nix~ and ~config.jsonc~.

[[https://github.com/fastfetch-cli/fastfetch][Fastfetch]] is a system information tool. We're already pulling it in via ~cli-tools.nix~, so this module just drops in the config.

* NixOS module

#+begin_src nix
{ ... }:
{
  flake.modules.homeManager.fastfetch =
  { ... }:
  {
    xdg.configFile."fastfetch/config.jsonc".source = ./config.jsonc;
  };
}
#+end_src
